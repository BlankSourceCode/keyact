# Keyact
<p align="center">

A typing trainer written in react/redux to help me learn to type in colemak-dh.

There is nothing specific to colemak-dh in the code. It is just a fun project to play around with.

# Building
1. `git clone` the project
1. `npm install`
1. `npm run start`
